# SH Test Pit Locations - Photo Evidence Tracking

Interactive Leaflet map showing test pit locations across multiple sites with embedded photo evidence.

## Live site

Once deployed, your site will be available at:
`https://<your-github-username>.github.io/<repo-name>/`

## What's in this folder

- `index.html` - The complete self-contained map application (all photos embedded as base64; no external assets needed)

## Notes

- Single-file deployment. The `Evidence/` folder is **not** required for the live site - all 174 photos are already embedded inside `index.html`.
- File size: ~77 MB (under GitHub's 100 MB hard limit).
